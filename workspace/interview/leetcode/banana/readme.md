简单查找 时间开销是n
二分查找的写法是有规律的
x  y 要找的是最小可以的min 可以来优化的
找中间，min = x + ((y - x) >> 1), 中间的小了,
mid  + 1 新的x
如果大了 mid - 1 新的y
